# Трекер настроения (Mood Tracker)

Консольное приложение на Python для ежедневного отслеживания и анализа настроения.

## О проекте

Программа помогает вести дневник эмоций в течение месяца. Каждый день пользователь оценивает своё настроение по шкале от 1 до 5. В конце программа показывает подробную статистику, визуализирует данные и сохраняет результат в файл.

## Возможности

- Сбор оценок за 31 день;
- Визуализация эмодзи (😊 🙂 😐 😕 😢);
- Вывод по дням с датами и графиком точками;
- Расчёт средней оценки;
- Определение лучшего и худшего дня;
- Подсчёт количества хороших, средних и плохих дней;
- Итоговый вердикт по результатам месяца;
- Автоматическое сохранение результатов в файл mood_history.txt.

## Стек технологий

- Python 3
- Стандартные библиотеки: datetime, timedelta

## Установка и запуск

1. Убедитесь, что на компьютере установлен Python (версия 3.6 или выше)
2. Скачайте репозиторий: 
https://github.com/Lymaralal/mood-tracker.git
3. Перейдите в папку проекта: 
cd mood-tracker
4. Запустите программу: 
python mood_tracker.py

## Как пользоваться

1. Введите своё имя;
2. Последовательно оцените каждый из 31 дня по шкале от 1 до 5: 
3. После ввода всех данных программа покажет:
- Эмодзи
- График по дням
- Статистику
- Вердикт
4. В папке с программой появится файл mood_history.txt с результатами
